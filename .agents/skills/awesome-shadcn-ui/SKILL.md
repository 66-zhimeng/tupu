## How to Use This Skill

When building a UI feature, search this list for relevant components:

1. **Check if a component exists** before building from scratch
2. **Prefer established libraries** (Magic UI, Aceternity, Origin UI) for animations
3. **Use v0.dev** for rapid prototyping with shadcn components
4. **Reference registries** (21st.dev) to discover new components
5. **Apply themes** using tweakcn or the theme generator for consistent styling
